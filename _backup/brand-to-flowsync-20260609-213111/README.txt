QRYX Static QR Code Generator

How to run:
1. Extract the zip.
2. Open index.html in your browser.

No backend. No server. No database. No permanent data storage.

Important:
- QR generation uses the qr-code-styling library from jsDelivr CDN.
- If you want fully offline use, download that library and replace the CDN script in index.html with a local file.

Real features included:
- Light/dark theme
- URL, text, Wi-Fi, email, phone, SMS, vCard, location QR payloads
- Color customization
- Dot/shape/eye style customization
- Logo upload
- PNG/SVG download
- Templates
- Current-session My Codes list only, not saved permanently

Not included because they require backend:
- User login
- Scan analytics
- Dynamic QR redirect links
- Cloud saving
- Payment/subscription
